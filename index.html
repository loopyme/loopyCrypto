LoopyCryptor API¶

All functions are implemented in Class:Cryptor, but we use lambda functions to wrapped up the Cryptor functions. So you can use it easier. Like this:

import loopyCryptor

# generate a pair of key
public_key, private_key = loopyCryptor.generate_RSA_key()

# encrypt
text = "I hate verilog."
cipher_byte = loopyCryptor.RSA_encrypt(text,public_key)

# decrypt
decrypt_text = loopyCryptor.RSA_decrypt(cipher_byte,private_key)
print(decrypt_text) # I hate verilog.

Cryptor is based on AES-CBC-16 and RSA_PKCS

loopyCryptor.Cryptor.Cryptor.AES_encrypt(text, key=None)¶

Encrypt: Encode the string into a byte-stream, then add it to a multiple of 16, then obtained a symmetric encryption key that is updated daily and then encrypt the string with the key.It is worth noting that ‘’ is used in the completion.

Parameters

  • text – str String to be encrypted

  • key – byte AES key

Returns

byte Encrypted byte stream
